# lc_1023: Camelcase Matching

**Topic**: [Two Pointers](../Two_Pointers.md)

**Empirical Time Complexity**: `O(N * L)`

**Empirical Space Complexity**: `O(N)`

## Notes

Both time and space complexity scale linearly with the number of queries N, where L is the maximum string length of the queries. The empirical space complexity accounts for the resulting boolean array of size N.

## Solution

```python
from typing import List

class Solution:
    def camelMatch(self, queries: List[str], pattern: str) -> List[bool]:
        res = []
        for query in queries:
            p = 0
            match = True
            for char in query:
                if p < len(pattern) and char == pattern[p]:
                    p += 1
                elif char.isupper():
                    match = False
                    break
            if match and p == len(pattern):
                res.append(True)
            else:
                res.append(False)
        return res

```
