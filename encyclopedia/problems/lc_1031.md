# lc_1031: Maximum Sum of Two Non-Overlapping Subarrays

**Topic**: [Sliding Window](../Sliding_Window.md)

**Empirical Time Complexity**: `O(N)`

**Empirical Space Complexity**: `O(1)`

## Notes

The timing ratio scales linearly as n increases from 10k to 100k, taking ~10x more time. The space complexity remains constant (~0.8 KB) across all scales since the algorithm only maintains running sums and pointers in O(1) auxiliary space.

## Solution

```python
from typing import List

class Solution:
    def maxSumTwoNoOverlap(self, nums: List[int], firstLen: int, secondLen: int) -> int:
        def helper(L: int, M: int) -> int:
            sumL = 0
            for i in range(L):
                sumL += nums[i]

            sumM = 0
            for i in range(L, L + M):
                sumM += nums[i]

            maxL = sumL
            ans = maxL + sumM

            for i in range(L + M, len(nums)):
                sumL += nums[i - M] - nums[i - M - L]
                maxL = max(maxL, sumL)

                sumM += nums[i] - nums[i - M]
                ans = max(ans, maxL + sumM)

            return ans

        return max(helper(firstLen, secondLen), helper(secondLen, firstLen))

```
