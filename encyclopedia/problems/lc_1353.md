# lc_1353: Maximum Number of Events That Can Be Attended

**Topic**: [Heap](../Heap.md)

**Empirical Time Complexity**: `O(N log N)`

**Empirical Space Complexity**: `O(N)`

## Notes

Time complexity is O(N log N) because sorting takes O(N log N) and heap operations take O(N log N) in worst case. Space is O(N) due to sorting and heap space. Timing ratio roughly scales by 10x-12x as n scales 10x, matching O(N log N).

## Solution

```python
from typing import List
import heapq

class Solution:
    def maxEvents(self, events: List[List[int]]) -> int:
        events.sort(key=lambda x: x[0])
        heap = []
        n = len(events)
        idx = 0
        d = 0
        ans = 0

        while idx < n or heap:
            if not heap and idx < n and d < events[idx][0]:
                d = events[idx][0]

            while idx < n and events[idx][0] == d:
                heapq.heappush(heap, events[idx][1])
                idx += 1

            while heap and heap[0] < d:
                heapq.heappop(heap)

            if heap:
                heapq.heappop(heap)
                ans += 1

            d += 1

        return ans

```
