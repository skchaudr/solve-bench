# lc_1238: Circular Permutation in Binary Representation

**Topic**: [Backtracking](../Backtracking.md)

**Empirical Time Complexity**: `O(N)`

**Empirical Space Complexity**: `O(N)`

## Notes

Here N refers to 2^n, which is the size of the output. The mapped n values are 7, 10, 13, and 17. The time and space grow linearly with 2^n (i.e. O(2^n) in terms of n, but O(N) in terms of benchmark scale N).

## Solution

```python
from typing import List

class Solution:
    def circularPermutation(self, n: int, start: int) -> List[int]:
        return [start ^ (i ^ (i >> 1)) for i in range(1 << n)]

```
