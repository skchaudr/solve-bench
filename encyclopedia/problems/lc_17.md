# lc_17: Letter Combinations of a Phone Number

**Topic**: [Backtracking](../Backtracking.md)

**Empirical Time Complexity**: `O(N * 4^N)`

**Empirical Space Complexity**: `O(N * 4^N)`

## Notes

To avoid OOM, standard N scales were mapped to actual sequence lengths of 6, 8, 10, and 12. Between N=6 to N=8 (ΔN=2), 4^2 = 16. The time ratio is ~30x and space ratio is ~16x. Between N=10 and N=12, the time ratio is ~15.7x and space ratio is ~16.6x. This confirms the O(N * 4^N) exponential time and space complexity due to generating combinations of size up to 4 characters per digit.

## Solution

```python
from typing import List

class Solution:
    def letterCombinations(self, digits: str) -> List[str]:
        if not digits:
            return []

        phone_map = {
            '2': 'abc', '3': 'def', '4': 'ghi', '5': 'jkl',
            '6': 'mno', '7': 'pqrs', '8': 'tuv', '9': 'wxyz'
        }
        res = []
        path = []

        def backtrack(index: int):
            if index == len(digits):
                res.append("".join(path))
                return

            for letter in phone_map[digits[index]]:
                path.append(letter)
                backtrack(index + 1)
                path.pop()

        backtrack(0)
        return res

```
