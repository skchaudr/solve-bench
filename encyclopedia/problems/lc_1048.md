# lc_1048: Longest String Chain

**Topic**: [Two Pointers](../Two_Pointers.md)

**Empirical Time Complexity**: `O(N log N)`

**Empirical Space Complexity**: `O(N)`

## Notes

Time complexity increases by factors of ~10.9x, ~10.5x, and ~11.7x for each 10x increase in N. This slightly supra-linear growth matches O(N log N), primarily driven by the initial sorting of words by length. Space complexity increases roughly linearly (~7.7x, ~7.9x, ~18.5x) corresponding to O(N) memory required for the DP hash map storing maximum chain lengths for each word.

## Solution

```python
from typing import List

class Solution:
    def longestStrChain(self, words: List[str]) -> int:
        words.sort(key=len)
        dp = {}
        max_chain = 1
        for word in words:
            dp[word] = 1
            for i in range(len(word)):
                prev_word = word[:i] + word[i+1:]
                if prev_word in dp:
                    dp[word] = max(dp[word], dp[prev_word] + 1)
            max_chain = max(max_chain, dp[word])
        return max_chain

```
