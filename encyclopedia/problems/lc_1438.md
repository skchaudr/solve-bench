# lc_1438: Longest Continuous Subarray With Absolute Diff Less Than or Equal to Limit

**Topic**: [Heap](../Heap.md)

**Empirical Time Complexity**: `O(N log N)`

**Empirical Space Complexity**: `O(N)`

## Notes

The timing ratio increases slightly faster than linear, consistent with an O(N log N) algorithm where each element is pushed to and lazily popped from a heap. The memory scales in proportion to the number of elements in the heap.

## Solution

```python
import heapq
from typing import List

class Solution:
    def longestSubarray(self, nums: List[int], limit: int) -> int:
        max_heap = []  # stores (-val, index)
        min_heap = []  # stores (val, index)
        left = 0
        ans = 0

        for right, val in enumerate(nums):
            heapq.heappush(max_heap, (-val, right))
            heapq.heappush(min_heap, (val, right))

            while -max_heap[0][0] - min_heap[0][0] > limit:
                left = min(max_heap[0][1], min_heap[0][1]) + 1
                while max_heap[0][1] < left:
                    heapq.heappop(max_heap)
                while min_heap[0][1] < left:
                    heapq.heappop(min_heap)

            ans = max(ans, right - left + 1)

        return ans

```
