# lc_1338: Reduce Array Size to The Half

**Topic**: [Heap](../Heap.md)

**Empirical Time Complexity**: `O(N log N)`

**Empirical Space Complexity**: `O(N)`

## Notes

Counting frequencies is O(N) time and O(U) space. Sorting the frequencies is O(U log U), where U is unique elements. Space is O(U) bounded by O(N).

## Solution

```python
class Solution:
    def minSetSize(self, arr: list[int]) -> int:
        from collections import Counter
        counts = Counter(arr)

        freqs = sorted(counts.values(), reverse=True)

        removed = 0
        set_size = 0
        half = len(arr) // 2

        for f in freqs:
            removed += f
            set_size += 1
            if removed >= half:
                break

        return set_size

```
