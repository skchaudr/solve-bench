# lc_1054: Distant Barcodes

**Topic**: [Heap](../Heap.md)

**Empirical Time Complexity**: `O(N log N)`

**Empirical Space Complexity**: `O(N)`

## Notes

Time complexity is bounded by O(N log N) since N heap operations take log(N) time each, and we observe approximately 10x-11x time increase for each 10x input size increase. Space complexity is O(N) as peak memory shows a 10x multiplier reflecting the storage for frequency maps, max heap, and output arrays.

## Solution

```python
from collections import Counter
import heapq
from typing import List

class Solution:
    def rearrangeBarcodes(self, barcodes: List[int]) -> List[int]:
        if not barcodes:
            return []

        freq = Counter(barcodes)
        max_heap = [[-count, val] for val, count in freq.items()]
        heapq.heapify(max_heap)

        res = []
        while len(max_heap) >= 2:
            count1, val1 = heapq.heappop(max_heap)
            count2, val2 = heapq.heappop(max_heap)

            res.append(val1)
            res.append(val2)

            if count1 + 1 < 0:
                heapq.heappush(max_heap, [count1 + 1, val1])
            if count2 + 1 < 0:
                heapq.heappush(max_heap, [count2 + 1, val2])

        if max_heap:
            res.append(max_heap[0][1])

        return res

```
