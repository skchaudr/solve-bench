# lc_1170: Compare Strings by Frequency of the Smallest Character

**Topic**: [Binary Search](../Binary_Search.md)

**Empirical Time Complexity**: `O(N log N)`

**Empirical Space Complexity**: `O(N)`

## Notes

Time complexity scales empirically at slightly more than O(N) due to sorting (O(N log N)). Memory complexity scales in O(N) due to list allocations scaling with N.

## Solution

```python
from typing import List
import bisect

class Solution:
    def numSmallerByFrequency(self, queries: List[str], words: List[str]) -> List[int]:
        def f(s: str) -> int:
            if not s:
                return 0
            min_char = min(s)
            return s.count(min_char)

        word_freqs = [f(w) for w in words]
        word_freqs.sort()

        ans = []
        n = len(word_freqs)
        for q in queries:
            freq = f(q)
            # Find the number of words with frequency strictly greater than `freq`
            idx = bisect.bisect_right(word_freqs, freq)
            ans.append(n - idx)

        return ans

```
