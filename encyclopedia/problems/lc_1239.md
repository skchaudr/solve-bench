# lc_1239: Maximum Length of a Concatenated String with Unique Characters

**Topic**: [Backtracking](../Backtracking.md)

**Empirical Time Complexity**: `O(N)`

**Empirical Space Complexity**: `O(N)`

## Notes

Time complexity is empirical O(N): A 10x increase in N (from 1k to 10k, and 10k to 100k) results in a ~7-8x increase in execution time. This happens because the heavy backtracking DFS operates on a bounded set of unique bitmasks (max 2^26), so the DFS time is O(1) relative to N. The dominant factor as N scales is the initial O(N) pass to parse the strings and populate the unique masks set. Space complexity is also O(N): peak memory increases by ~5-10x for every 10x increase in N, primarily due to the storage of the input array and the set of unique masks.

## Solution

```python
from typing import List

class Solution:
    def maxLength(self, arr: List[str]) -> int:
        unique_masks = set()
        for s in arr:
            mask = 0
            dupes = False
            for c in s:
                bit = 1 << (ord(c) - 97)
                if mask & bit:
                    dupes = True
                    break
                mask |= bit
            if not dupes:
                unique_masks.add(mask)

        # Sort masks by length descending for better pruning
        masks = sorted(list(unique_masks), key=lambda x: -x.bit_count())
        n = len(masks)

        # Precompute suffix bitwise ORs for pruning
        suffix_masks = [0] * n
        curr_or = 0
        for i in range(n - 1, -1, -1):
            curr_or |= masks[i]
            suffix_masks[i] = curr_or

        max_len = [0]

        def dfs(idx, current_mask, current_len):
            if current_len > max_len[0]:
                max_len[0] = current_len

            if idx == n:
                return

            # Pruning:
            # If the current bits we have + the best we can possibly add from here
            # is less than or equal to max_len[0], we prune.
            # The best we can add is bounded by the bits available in suffix_masks[idx]
            # that don't conflict with current_mask.
            available_bits = suffix_masks[idx] & ~current_mask
            if current_len + available_bits.bit_count() <= max_len[0]:
                return

            for i in range(idx, n):
                mask = masks[i]
                if (current_mask & mask) == 0:
                    dfs(i + 1, current_mask | mask, current_len + mask.bit_count())

        dfs(0, 0, 0)
        return max_len[0]

```
