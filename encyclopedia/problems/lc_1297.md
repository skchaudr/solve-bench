# lc_1297: Maximum Number of Occurrences of a Substring

**Topic**: [Sliding Window](../Sliding_Window.md)

**Empirical Time Complexity**: `O(N)`

**Empirical Space Complexity**: `O(1)`

## Notes

Time complexity scales roughly linearly as n increases by a factor of 10. The time complexity is bounded by O(N * minSize) and since minSize <= 26, it reduces to O(N). Space complexity is theoretically O(N) or bounded by the maximum number of unique strings of length minSize (which is a constant at most 26^minSize), as evidenced by the flat/decreasing memory footprint.

## Solution

```python
class Solution:
    def maxFreq(self, s: str, maxLetters: int, minSize: int, maxSize: int) -> int:
        counts = {}
        max_freq = 0

        for i in range(len(s) - minSize + 1):
            sub = s[i:i+minSize]
            if len(set(sub)) <= maxLetters:
                counts[sub] = counts.get(sub, 0) + 1
                if counts[sub] > max_freq:
                    max_freq = counts[sub]

        return max_freq

```
