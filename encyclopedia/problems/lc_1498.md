# lc_1498: Number of Subsequences That Satisfy the Given Sum Condition

**Topic**: [Two Pointers](../Two_Pointers.md)

**Empirical Time Complexity**: `O(N log N)`

**Empirical Space Complexity**: `O(N)`

## Notes

The time complexity is O(N log N) because of the initial sorting of the array. Precomputing powers of 2 takes O(N) time, and the two-pointer traversal takes O(N) time. Overall, sorting dominates. The 10x increases in N roughly lead to slightly more than 10x increases in time (from 118ms to 1247ms), which aligns well with O(N log N) empirical measurements. The space complexity is O(N) due to the array used for precomputing powers of 2 (peak memory scales linearly: 4 KB -> 39 KB -> 390 KB -> 3906 KB).

## Solution

```python
from typing import List

class Solution:
    def numSubseq(self, nums: List[int], target: int) -> int:
        nums.sort()
        n = len(nums)
        MOD = 10**9 + 7

        pow2 = [1] * n
        for i in range(1, n):
            pow2[i] = (pow2[i - 1] * 2) % MOD

        res = 0
        l, r = 0, n - 1
        while l <= r:
            if nums[l] + nums[r] <= target:
                res = (res + pow2[r - l]) % MOD
                l += 1
            else:
                r -= 1

        return res

```
