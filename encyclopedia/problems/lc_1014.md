# lc_1014: Best Sightseeing Pair

**Topic**: [Dynamic Programming](../Dynamic_Programming.md)

**Empirical Time Complexity**: `O(N)`

**Empirical Space Complexity**: `O(1)`

## Notes

Time complexity scales linearly with N. Time increases approximately 10x for each 10x increase in N (e.g., from ~1.85 ms to ~18.89 ms). Space complexity is strictly O(1) auxiliary space, as tracemalloc reflects constant memory (~0.26 KB) irrespective of input scale N.

## Solution

```python
from typing import List

class Solution:
    def maxScoreSightseeingPair(self, values: List[int]) -> int:
        max_i_val = values[0]
        max_score = 0

        for j in range(1, len(values)):
            score = max_i_val + values[j] - j
            if score > max_score:
                max_score = score

            i_val = values[j] + j
            if i_val > max_i_val:
                max_i_val = i_val

        return max_score

```
