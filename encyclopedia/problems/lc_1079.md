# lc_1079: Letter Tile Possibilities

**Topic**: [Backtracking](../Backtracking.md)

**Empirical Time Complexity**: `O(N!)`

**Empirical Space Complexity**: `O(N)`

## Notes

Backtracking explores all unique subsets/permutations, leading to factorial time complexity. Space complexity is O(N) due to recursion stack depth and character frequencies. N values mapped down to 3, 5, 7, 9 to avoid timeout.

## Solution

```python
class Solution:
    def numTilePossibilities(self, tiles: str) -> int:
        from collections import Counter

        counts = Counter(tiles)

        def backtrack():
            total = 0
            for char in counts:
                if counts[char] > 0:
                    total += 1
                    counts[char] -= 1
                    total += backtrack()
                    counts[char] += 1
            return total

        return backtrack()

```
