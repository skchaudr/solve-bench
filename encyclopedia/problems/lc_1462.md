# lc_1462: Course Schedule IV

**Topic**: [Graph](../Graph.md)

**Empirical Time Complexity**: `O(V * E)`

**Empirical Space Complexity**: `O(V^2)`

## Notes

Time scales empirically around O(N log N) to O(N^1.5) on this sparse dataset, but worst-case time is O(V^3) or O(V*E) without bitsets. With large ints, it's effectively O(V*E / 64) -> O(N^2) since E~V. Space clearly scales quadratically (10x N -> ~80x memory), so O(V^2).

## Solution

```python
from typing import List

class Solution:
    def checkIfPrerequisite(self, numCourses: int, prerequisites: List[List[int]], queries: List[List[int]]) -> List[bool]:
        adj = [[] for _ in range(numCourses)]
        in_degree = [0] * numCourses
        for u, v in prerequisites:
            adj[u].append(v)
            in_degree[v] += 1

        q = [i for i in range(numCourses) if in_degree[i] == 0]

        ancestors = [1 << i for i in range(numCourses)]

        head = 0
        while head < len(q):
            u = q[head]
            head += 1
            for v in adj[u]:
                ancestors[v] |= ancestors[u]
                in_degree[v] -= 1
                if in_degree[v] == 0:
                    q.append(v)

        return [(ancestors[v] & (1 << u)) != 0 for u, v in queries]

```
