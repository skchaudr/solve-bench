# lc_1248: Count Number of Nice Subarrays

**Topic**: [Sliding Window](../Sliding_Window.md)

**Empirical Time Complexity**: `O(N)`

**Empirical Space Complexity**: `O(1)`

## Notes

The timing ratio increases by approximately 10x for each 10x increase in n, indicating a linear O(N) time complexity. The peak memory usage remains constant around 0.4 KB regardless of the input size, indicating O(1) space complexity.

## Solution

```python
from typing import List

class Solution:
    def numberOfSubarrays(self, nums: List[int], k: int) -> int:
        def atMost(k: int) -> int:
            if k < 0:
                return 0
            ans = 0
            left = 0
            for right in range(len(nums)):
                k -= nums[right] % 2
                while k < 0:
                    k += nums[left] % 2
                    left += 1
                ans += right - left + 1
            return ans

        return atMost(k) - atMost(k - 1)

```
