# lc_1237: Find Positive Integer Solution for a Given Equation

**Topic**: [Two Pointers](../Two_Pointers.md)

**Empirical Time Complexity**: `O(N)`

**Empirical Space Complexity**: `O(N)`

## Notes

Both time and space scale linearly by roughly 10x for each 10x increase in n. The linear space complexity is driven by the output size; there are exactly n-1 valid pairs added to the result list in our stress test data, meaning O(N) space is required to store the answer.

## Solution

```python
from typing import List

class CustomFunction:
    # Interface only for type hinting in Solution
    def f(self, x: int, y: int) -> int:
        pass

class Solution:
    def findSolution(self, customfunction: CustomFunction, z: int) -> List[List[int]]:
        res = []
        x = 1
        y = z
        while x <= z and y >= 1:
            val = customfunction.f(x, y)
            if val == z:
                res.append([x, y])
                x += 1
                y -= 1
            elif val < z:
                x += 1
            else:
                y -= 1
        return res

```
